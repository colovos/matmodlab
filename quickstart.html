Quick Start Guide¶

The steps for running a simulation in matmodlab are

  • Setup and Build
  • Prepare Input
  • Run the Simulation
  • Postprocess

Setup and Build¶

The following commands are to be executed from a command prompt

Build matmodlab. See Building* for more details.

  • Clone matmodlab from https://github.com/tjfulle/matmodlab:

    git clone https://github.com/tjfulle/matmodlab.git

  • Navigate to the matmodlab directory and execute:

    python setup.py develop

All components of matmodlab will be built and installed.

Prepare Input¶

Input files are Python scripts. See Annotated Examples and The Material Point Simulator* for more details.

  • Instantiate a MaterialPointSimulator object.
  • Define the material model
  • Define the deformation path

Run the Simulation¶

Run the input script with mml. See Annotated Examples for more details:

mml run [options] filename.py

mml will create the following files:

ls filename.*
filename.exo            filename.log            filename.py

filename.exo is the ExodusII output database and filename.log the simulation log file. See mml help run for a complete list of options

Postprocess¶

View results in matmodlab viewer. See Postprocessing Results for more details:

mml view filename.exo
